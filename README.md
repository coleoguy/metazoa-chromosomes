# CURE-chromosome-number-evolution

The in this repository recapitulate analyss of chromosome Evolution
CURE Project. This project investigates rate variation in fusion, fission, demiploidy,
and whole genome duplication.

1.  README.md: The file that you are reading
2.  **data**
3.  chrome: contains csv files with phenotype data for all clades
4.  trees: contains phylogenies for all clades
5.  **script**: contains all scripts for data validation and analysis
6.  **results**: contains all results
7.  **figures**: contains scripts to produce each figure.
